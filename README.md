# TinyApp Project

TinyApp is a full stack web application built with Node and Express that allows users to shorten long URLs (Ã  la bit.ly).
<br>
This is my second project at LHL after [Lotide](https://github.com/thaian161/lotide) - a mini clone of [Lodash](https://lodash.com) library.
<br>
<br>
Once ***creating an account*** and ***logging in*** TinyApp, users can ***shorten***, ***edit*** and ***delete*** URLs . User accounts are protected with ***cookies and password encryption*** using cookies-session, bcrypt, hash, and salt. This ensures that only authenticated users can access their URL storage.

## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session
- morgan

## Dev-Dependencies

- chai
- mocha
- nodemon

## Getting Started

- Install all dependencies (using the `npm install` command).
- Run the development web server using the `node express_server.js` command.
